A simple WordPress plugin designed to extend the functionality of your theme and provide seamless integration between the theme and the Contact Form 7 plugin. It enhances the capabilities of your theme by providing additional features and options for contact forms created using Contact Form 7.

== Description ==

### WCA: Contact Form 7: the #1 WeCodeArt Framework + Contact Form 7 plugin
A simple extension for [WeCodeArt Framework](https://wordpress.org/themes/wecodeart/) used as a bridge between Contact Form 7 and our theme.

== Installation ==
Starting with WCA: Contact Form 7 consists of just two steps: installing and setting up the plugin.

### INSTALL WCA: Contact Form 7 FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’;
1. Search for ‘WCA: Contact Form 7’;
1. Activate WCA: Contact Form 7 from your Plugins page;
1. Go to ‘after activation’ below.

### INSTALL WCA: Contact Form 7 MANUALLY

1. Upload the ‘wca-contact-form-7’ folder to the /wp-content/plugins/ directory;
1. Activate the WCA: Contact Form 7 plugin through the ‘Plugins’ menu in WordPress;
1. Go to ‘after activation’ below.

### AFTER ACTIVATION

1. You should see (a notice to start) about WCA: Contact Form 7 activation;
1. Go through Appearance -> Theme Options -> Extensions -> Contact Form 7 and set up the plugin for your site;
1. You’re done!

== Frequently Asked Questions ==

= Why this plugin can't be activated? =

You are probably not running [WeCodeArt Framework](https://wordpress.org/themes/wecodeart/) or a Child Theme of it.

= Where are the plugin options located? =

The options are located under Extensions panel in Appearance -> Theme Options -> Extensions.
